# Source Code

This directory contains the main source code for the Python Refactor MCP server.

## Structure

- `python_refactor_mcp/` - Main package implementing the MCP-LSP bridge

## Overview

The source code implements a Model Context Protocol (MCP) server that bridges to the Pyright Language Server Protocol (LSP) server, enabling AI agents to perform semantically-aware Python refactoring operations.

See the main [README.md](../README.md) for full documentation.
